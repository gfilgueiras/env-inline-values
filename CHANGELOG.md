## [0.1.0] - 2025-12-23

### Added

- Initial release of **Env Inline Values**
- Inline display of resolved `.env` variable values in the editor
- Support for nested variable resolution (e.g. `VAR2=${VAR3}`)
- Automatic updates when files or `.env` change
- Works across any file type (Dockerfile, YAML, Makefile, Shell scripts, `.env`, etc.)
- Cross-platform support (Windows, Linux, macOS)
